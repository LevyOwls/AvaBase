Requerimientos previos

  • Para poder utilizar 'Ava js' se debe contar con:
    • Node js^12.x

    Comandos de instalación

    Para instalar Ava js, basta con utilizar el siguiente comando en la carpeta donde se quiera instalar:

    npm init ava

    Agregando selenium

    Para poder trabajar con selenium, se debe instalar y para ello se ingresa el siguiente comando

    npm install selenium-webdriver chromedriver

    Agregando supertest

    Para utilizar supertest con Ava js, se debe ingresar el siguiente comando

    npm install supertest --save-dev

    Sintaxis

    La sintaxis es sencilla, por lo cual el unico conocimiento previo requerido es tener conocimiento basico de javascript

    Consideraciones

    Para poder realizar los tests, estos deben crearse y para que Ava js los reconozca deben tener la terminación:

    test.js

    Ejemplos de esto son:

    • unit_test.test.js
    • example.test.js
    • test.test.js
    • etc.etc.test.js

    Hola mundo

    El siguiente codigo es un ejemplo de un hola mundo


    //Se utiliza 'test' proveniente de ava
const test = require("ava");
//Se realiza un test el cual solo consiste en pasar
test('foo', t => {
  t.pass();
});
//Se realiza un test con promesas, el cual tambien pasará exitosamente
test('bar', async t => {
  const bar = Promise.resolve('bar');
    t.is(await bar, 'bar');
});

    Para ejecutar el codigo, basta con escribir en consola el comando

    npm test

    Existen variaciones, entre los casos especiales se puede mencionar

    const test = require("ava");
//Se utiliza test.serial para ejecutar un test a la vez, de otra forma los ejecuta todos al mismo tiempo.
test.serial('foo', t => {
  t.pass();
});

//Utilizat test.skip resultara en que se omitira el test
test.skip('bar', t => {
  t.pass();

    Hola mundo, selenium

    El siguiente codigo es un ejemplo del uso de selenium en conjunto de ava

    //Se utiliza 'test' proveniente de ava
const test = require("ava");
//Se obtienen lo necesario para utilizar selenium
const {Builder, By, Key, until} = require('selenium-webdriver');
require('chromedriver');

//Antes de cada test, se crea una instancia de chrome y se abre la pagina de google
test.beforeEach(async t => {
  t.context.driver = new Builder().forBrowser('chrome').build();
  await t.context.driver.get('https://www.google.com');
});

//Cada vez que finalice un test, cierra la instancia de chrome
test.afterEach('cleanup', async t => {
  await t.context.driver.close();
});

//Funcion la cual ingresa una busqueda 
async function searchGoogle(driver, keyword) {
  await driver.findElement(By.name('q')).sendKeys(keyword + Key.ENTER);
  await driver.wait(until.titleIs(`${keyword} - Buscar con Google`));
}

//Test en el cual se obtiene la instancia de chrome, busca "social-it" y verifica que encontro resultados
test.serial('Googlear social-it', async t => {
  const {driver} = t.context;
  await searchGoogle(driver, 'social-it');
  t.true((await driver.findElement(By.id('result-stats')).getText()).includes('resultados'));
});

    Hola mundo, supertest

    Para realizar pruebas con supertest, se requiere:

    • Instalar Ava en la carpeta en la cual se esta trabajando
    • Un servidor, el cual se encuentre funcionando y recibiendo solicitudes, en este caso utilizaremos express

    Ya que se utilizará express, se creara el archivo app.js en la carpeta raíz, el cual se vera de la siguiente forma

    //Se importa lo necesario para que funcione express
const express = require('express')
//Se crea una app con express
const app = express()

//Se asigna la ruta "/" a app, lo que retorna "Hello World" si se accede a la ruta.
app.get('/', function (req, res) {
  res.send('Hello World')
})

//Inicia el servidor en el puerto 3000
app.listen(3000)
//Exporta el módulo app
module.exports = app

    El archivo test.js pasará a ser el siguiente

    //Se utiliza 'test' proveniente de ava
const test = require("ava");
//Se utiliza request para realizar peticiones, proveniente de supertest
const request = require("supertest");

//Crea el test 
test.serial('test express', async t => {
    //Llama a app
    const app = require('./app.js')
    //Realiza una solicitud 'get' al servidor
    const response = await request(app)
                    .get("/");
    //Imprime lo que retorna la solicitud
    console.log(response.text);
    //Verifica que el status de la respuesta sea 200
    t.is(response.status, 200);
});
